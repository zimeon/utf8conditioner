3.5  Impossible bytes                                                         |
                                                                              |
The following two bytes cannot appear in a correct UTF-8 string               |
                                                                              |
3.5.1  fe = "ş"                                                               |
3.5.2  ff = "ÿ"                                                               |
3.5.3  fe fe ff ff = "şşÿÿ"                                                   |
